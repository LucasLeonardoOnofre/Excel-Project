Bike Buyers Excel Dashboard Project

This project analyzes a Bike Buyers dataset using Microsoft Excel, focusing on data cleaning, analysis, and interactive visualization.

Overview

Created a working sheet to preserve raw data

Cleaned the dataset by removing duplicates, trimming spaces, standardizing text, formatting currencies, and removing unnecessary data

Prepared the data for analysis

Analysis & Visualization

Built three pivot tables:

Average income per purchase

Customer commute distance

Customer age brackets

Created three charts based on the pivot tables

Designed an interactive dashboard using the charts

Dashboard Features

Three interactive slicers

Dynamic charts that update based on slicer selections

Tools Used

Microsoft Excel (Pivot Tables, Charts, Slicers)

This project demonstrates an end-to-end Excel data analysis workflow from raw data cleaning to an interactive dashboard.
